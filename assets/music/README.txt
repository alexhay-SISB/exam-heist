PLACE YOUR BACKGROUND MUSIC FILE HERE
======================================

To enable background music in the game, add a file at:
  public/assets/music/heist-theme.mp3
  (or .ogg as fallback)

RECOMMENDED FREE MUSIC SOURCES (royalty-free, CC0/CC-BY):

1. **Pixabay Music** (https://pixabay.com/music/)
   Search: "spy", "heist", "tension", "mission impossible"
   No attribution required.

2. **OpenGameArt.org** (https://opengameart.org/art-search-advanced?keys=heist&type=0%5B%5D=12&sort_by=count&sort_order=DESC)
   Search: "heist", "spy", "stealth"

3. **Free Music Archive** (https://freemusicarchive.org/)

4. **YouTube Audio Library** (https://studio.youtube.com - if you have a Google account)

5. **Incompetech (Kevin MacLeod)** (https://incompetech.com/)
   Search: "Sneaky Snitch", "Spy Glass", "The Cannery"

If no file is present, the game will run silently (sound effects still work).

NOTE: Make sure any music you add is royalty-free and you have permission to use it.
